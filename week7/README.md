# Materials for week 7
- **corrosion_demo_figures/** — Directory containing images and figures used in the corrosion notebook
- **pump_datasheet/** — Directory containing data and figures used in the pump notebook
- **Supplementary/** — Directory containing supplementary material on adaptive sampling
- **synthetic_corrosion_data/** — Directory containing synthetic data and figures used in the corrosion notebook
- **corrosion_surrogates_demo.ipynb** — Notebook demonstrating how to build surrogate models for general and localized corrosion
- **fit_surrogates.py** — Helper function to fit the surrogates with adaptive sampling
- **mvc_corrosion.py** — File containing functions to run through the workflow of corrosion_surrogates_demo.ipynb
- **pump_surrogate_psymo_final.ipynb** — Notebook demonstrating how to build a surrogate model of an RO multispeed pump performance curve using PySMO
