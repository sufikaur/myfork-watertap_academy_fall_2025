# Materials for week 5
This week covers parameter sweep analysis using WaterTAP.

## Files

- **graphics/** — Directory containing images and figures used in the notebooks
- **RO_with_ERD.py** — Reverse osmosis flowsheet module with energy recovery device (ERD) for use in parameter sweep demonstration
- **lsrro.ipynb** — Notebook demonstrating Low Salt Rejection Reverse Osmosis (LSRRO) optimization and sensitivity analysis
- **parameter_sweep_demo.ipynb** — Notebook introducing the `parameter sweep` tool for systematic exploration of model behavior across parameter ranges
- **plot_functions.py** — Utility functions for visualizing parameter sweep results and flowsheet outputs
