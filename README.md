# WaterTAP Academy Fall 2025

Materials for each week are under `week<N>` directories.

The directories `src` and `data` can be used for storing any source code and data.

In order to run WaterTAP models, please first install the [WaterTAP main library](https://github.com/watertap-org/watertap).


### Explore the academy online with mybinder.org

Binder is an online service providing a short-lived temporary sandbox environment on public cloud resources where Jupyter notebooks can be run, free of charge, without having to install any software locally.

Click on this button to launch an environment pointing to the current main branch of this WaterTAP Academy repository:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/watertap-org/watertap_academy_fall_2025/main)
