# Materials for week 3

* **`intro_to_costing.ipynb`:** introduction for adding costing into an RO unit model.
* **`RO_Flowsheet_ERD_Techno_Economics_Analysis.ipynb`:**  Techno-Economic Analysis and Cost Optimization of an RO with ERD Flowsheet.
* **`utility_functions.py`:** helper functions used in `RO_Flowsheet_ERD_Techno_Economics_Analysis.ipynb` to help report results.

Post-session Tutorial:
* **`modeling_multicomponent_rejection.ipynb`:** another flowsheet example with costing. This example introduces the multicomponent aqueous solution (MCAS) property model and the Donnan Steric Pore Model with Dielectric Exclusion (DSPM-DE) for multicomponent rejection predictions with nanofiltration.
