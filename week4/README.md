# Materials for week 4

* **`creating_a_custom_costing_model.ipynb`:** workflow for adding custom costing relationships to a unit model
* **`creating_a_custom_property_model.ipynb`:** workflow for creating a custom property package
* **`creating_a_custom_unit_model.ipynb`:** workflow for creating a custom Filtration unit model
* **`creating_a_translator_block.ipynb`:** introduction to translator blocks and how to create them
* **`custom_cost_model.py`:** Python file recreating the cost model from `creating_a_custom_costing_model.ipynb`
* **`custom_prop_model.py`:** Python file recreating the property model from `creating_a_custom_property_model.ipynb`
* **`custom_unit_model.py`:** Python file recreating the unit model from `creating_a_custom_unit_model.ipynb`
* **`exporting_flowsheets_to_gui.ipynb`:** workflow for adding a new flowsheet to the WaterTAP GUI (IDAES Flowsheet Processor)
* **`filtration_RO_with_ERD.py`:** flowsheet connecting the custom Filtration models to the RO w/ERD flowsheet from class
* **`filtration_RO_with_ERD_ui.py`:** GUI export file that sends `filtration_RO_with_ERD.py` to the GUI
* **`setup.py`:** Setup file that allows the GUI to import flowsheets from the WaterTAP Academy repository
* **`translator.py`:** Python file recreating the translator unit model from `creating_a_translator_block.ipynb`
* **`using_custom_unit_and_prop_model.ipynb`:** workflow for integrating custom models into an existing flowsheet
